I can has iCal est une application web qui permet aux étudiants de l'École Polytechnique de Montréal d'avoir accès à une copie de leur horaire sous le format iCalendar.

Le format iCalendar est un format standard d'agenda vastement supporté. Plusieurs applications très connues telles Outlook et Google Calendar permettent d'importer des calendriers enregistrés sous ce format.

Il est également possible d'importer un fichier iCal à l'aide d'un iPod touch / iPhone permettant ainsi d'avoir une copie de son horaire dans sa poche en tout temps!
